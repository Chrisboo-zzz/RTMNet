# RTMNet: A Real-Time Turbulence Mitigation Network for Photogrammetric Structural Displacement Measurement

**Abstract:** Vision-based structural displacement measurement in field environments is frequently affected by atmospheric turbulence, which induces local geometric distortion, spatially varying blur, and temporal jitter, thereby compromising feature localization stability and displacement estimation reliability. Existing methods primarily focus on visual restoration quality but lack explicit constraints on measurement-related geometric fidelity. Moreover, real-time single-frame restoration fails to fully exploit short-term degradation history, thereby limiting online measurement accuracy. To address these limitations, this paper proposes RTMNet, a real-time turbulence mitigation network for photogrammetric structural displacement measurement. RTMNet employs Causal Spatio-Temporal Memory Blocks to recursively leverage short-term historical information and suppress turbulence-induced inter-frame jitter and local drift. A Latent Phase Distortion prior branch enhances the modeling of spatially non-uniform turbulence degradation, while measurement-oriented sparse geometric consistency supervision constrains the recovery of edge intersections, texture boundaries, and structural feature neighborhoods. Experimental results demonstrate that RTMNet reduces displacement measurement errors by 45.7% and 54.6% on the simulated dataset and laboratory turbulence dataset, respectively, and outperforms existing comparable models in both measurement accuracy and real-time inference efficiency. Tests on real turbulence-degraded structural scenes further indicate that RTMNet better preserves structural edges and local feature trajectories in previously unseen scenarios. These results highlight that RTMNet provides a high-precision, real-time turbulence mitigation solution for photogrammetric structural displacement monitoring under complex atmospheric conditions.
